One-Line Switch Logic

MECHANISM PLUG-IN SYSTEM
(One-Line Switch for All Prompts)

PRODUCT MECHANISM:
This product helps a mother identify which prerequisite concept a child is missing before understanding a lesson.

PURPOSE
This system allows you to reuse the SAME prompts while switching the product mechanism using ONE line only.
Truth Layer and Language Layer remain unchanged.
Only the Thinking Layer adapts automatically based on the mechanism line.

==================================================
HOW TO USE (IMPORTANT)
At the VERY TOP of any prompt, add ONE line:
PRODUCT MECHANISM: [WRITE ONE CLEAR SENTENCE HERE]

==================================================
MECHANISM INTERPRETATION RULE (MANDATORY)
"All explanations, stories, logic, and examples MUST be built ONLY around this mechanism. Do not assume any other mechanism. Do not reuse language from previous mechanisms."

==================================================
THINKING LAYER – DYNAMIC VERSION
THINKING LAYER (MECHANISM-DRIVEN)
* Do NOT assume how the product works.
* Read the PRODUCT MECHANISM line carefully.
* Identify:
  1. What problem exists BEFORE use
  2. Why it exists ACCORDING TO THE MECHANISM
  3. How product resolves it USING ONLY THE MECHANISM
* Build strictly from the mechanism (Diagnostic, Explanatory, Practice, or Guidance).
