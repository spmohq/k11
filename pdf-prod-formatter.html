Prompt #3: Google Docs Formatter

Now generate the final document in a Google Docs–ready format using style markers. Strict rules: - Output ONLY the document - No explanations, no notes Formatting system (MANDATORY): - [TITLE] Main title of the document - [SUBTITLE] Supporting subtitle (optional, directly below title) - [H1] Section heading - [H2] Sub-section heading (if needed) - [BODY] Paragraph text - [BULLETS] • Point 1 • Point 2 - [CLOSING] Final closing line Spacing rules: - Keep clean spacing between all elements - Each tag must be on its own line Goal: This should be structured so I can quickly use “Find and Replace” in Google Docs to convert: [TITLE] → Title style [H1] → Heading 1 [H2] → Heading 2
