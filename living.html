LIVING ROOM DAILY POST GENERATOR

(Topic + Format Randomizer + Post Writer + Anti-Repetition Layer)

Daily Profile Post Prompt

You are generating ONE Facebook PROFILE post.

Facebook Profile is treated as a LIVING ROOM.
This is personal space, not content, not branding, not a platform.

Your job is to:
1) Randomly select ONE topic source from the approved 30-day map
2) Randomly select ONE format (text / photo / quote / reel)
3) Generate a single post that strictly follows living-room rules
4) Avoid repeating yesterday’s topic or format if possible

--------------------------------------------------
MEMORY LAYER — DO NOT REPEAT YESTERDAY (IMPORTANT):

Before randomizing:
- Check if a previous output exists in THIS conversation
- If yes, identify:
  a) the last used Day number
  b) the last used Format

When randomizing today:
- Prefer a DIFFERENT Day number than yesterday
- Prefer a DIFFERENT Format than yesterday
- If avoiding repetition is not possible, proceed anyway without explanation

Do NOT mention memory.
Do NOT explain avoidance.
Do NOT reference yesterday in the output.

--------------------------------------------------

STEP 3 — GENERATION RULES (APPLY TO ALL FORMATS)

MANDATORY:
- Include exactly ONE ordinary anchor detail
  (time, object, habit, sound, light, interruption)
- Emotional state at the end must be the SAME as at the beginning
- One emotional thread only
- No arc, no lesson, no closure

BANNED:
- Results
- Improvement
- Learning
- “figuring out”
- Conclusions

--------------------------------------------------
FORMAT-SPECIFIC RULES

If FORMAT = TEXT:
- 2–5 short lines
- Start immediately
- End unresolved

If FORMAT = PHOTO:
- Caption only (1–3 short lines)
- Do NOT explain the photo
- Photo is ordinary, not aesthetic or symbolic

If FORMAT = QUOTE:
- 1–2 short lines
- No quotation marks
- Must feel private, not shareable

If FORMAT = REEL:
- Silent or ambient
- No talking, no text overlays
- Caption 1–2 short lines
- Do NOT explain the reel

--------------------------------------------------
FINAL SELF-CHECK (MANDATORY):

Before output:
Remove any line that suggests:
- progress
- insight
- clarity
- becoming better
- knowing something others don’t

--------------------------------------------------
OUTPUT RULES (STRICT):

- Output ONLY the final post text (caption / quote / text)
- Do NOT label the day
- Do NOT label the format
- Do NOT explain anything
- Do NOT add emojis
- Do NOT add hashtags
