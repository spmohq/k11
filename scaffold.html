System-to-Scaffold Prompt Chain

A structured prompt chain that turns any complex system or document into a clear, categorized, step-by-step learning and execution scaffold.

Prompt #1: System Analysis & Understanding

Please read, understand, and review the uploaded document.

Your response must include the following three elements:
A comprehensive understanding of the system, including:
- How the system functions
- The chronological order of its steps
- The purpose of each step
- How each step contributes to the overall system

A detailed operational guide describing, for each step:
- How the step is executed
- The required inputs
- The expected outputs

A structured organization of activities presented in the following format: [Activity Type / Name]: Activity Index

Use clear headings and concise, instructional language. Avoid unnecessary repetition.

Prompt #2: Sequential Guidance Mode

Now guide and help me with this, but do it slowly, one step at a time.

Prompt #3: Categorized Activity List

Before that, give me a step-by-step bullet list of all the activities I need to do. Categorize them as well.

Prompt #4: Slow Execution Handoff

Now I’m ready to proceed but slowly, one step at a time. 

1. Give me a categorized bullet list of all the activities I need to do, step by step.
2. Then guide and help me through each step slowly, one step at a time.

Prompt #5: Standalone Template Creation

Use [placeholder] to design a standalone, reusable prompt template. The template must fully integrate all required context, assumptions, constraints, and step-by-step instructions necessary for ChatGPT to complete this step independently. The output should be self-contained, so that when the template is used in a new chat, it reproduces all required information accurately and completely. After this is completed, we will proceed to the next step.

Prompt #6: Chain Consolidation

You are familiar with all the prompts from 1 to 20. Please treat them as a prompt chain, name it, and provide a brief description.

Prompt #7: Positioning & Transformation

What problem does it solve, and who suffers from this problem? Please state their before-and-after conditions.

Prompt #8: Summary & Purpose Index

Generate a bullet list of all steps. For each step, include the step number, step name, and the purpose of the step expressed as verb + verb’s object.
