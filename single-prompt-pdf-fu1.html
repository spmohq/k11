Single Prompt Follow up #1

Using the product framing, positioning, and tone already established, write the COMPLETE content of the PDF for a SINGLE prompt product.

Assume:
- This product contains ONE prompt only
- It is designed to solve ONE clear problem
- All positioning and audience decisions are already locked

Do NOT reframe this as a system, bundle, or framework.

The PDF must contain ONLY the following five sections, in this exact order, with no section titles:

1. Overall introduction  
- Calm, grounded, and specific  
- Explain the single problem this prompt helps solve  
- Set realistic expectations  
- Emphasize simplicity and focus  

2. The prompt  
- Present the prompt cleanly and clearly  
- No commentary inside the prompt text  
- No variations or alternatives  

3. How to use this prompt  
- Step-by-step usage  
- Minimal, practical, and non-technical  
- Explain how to run it and what to adjust  

4. Where to use the outcome of this prompt and how  
- Specific contexts  
- Explain how to apply the output, not just where  

5. “Before you go”  
- Short, grounded closing note  
- Reinforce that this is a small, reusable tool  

Strict rules:
- No hype, no emojis, no inflated promises  
- No references to Gumroad, pricing, or selling  
- No “system”, “framework”, or “bundle” language  
- Language must feel intentional and paid  

Write clean paragraphs.
Short sentences.
Clear structure.

Now write the complete PDF content.
