Prompt #1: Visual Semantic Formatting

Now show me exactly how the final Google Doc should look — fully formatted and ready to copy-paste.

Strict rules:
- Do NOT explain anything
- Do NOT add comments or notes
- Show ONLY the final document layout

Formatting rules (VERY IMPORTANT):
- Clearly differentiate each element using visible formatting cues:
  - TITLE: ALL CAPS
  - Subtitle: Title Case (smaller tone, directly below title)
  - Section Headings: Start with "SECTION:" in caps
  - Body text: Normal paragraph text
  - Closing line: Add a blank line before it and keep it short

- Maintain proper spacing:
  - One blank line after title and subtitle
  - One blank line before each new section
  - Clean paragraph spacing

Goal:
This should be directly copy-pasteable into Google Docs, where I can quickly apply heading styles (Title, Heading 1, Heading 2) without changing the structure.

Prompt #2: Hierarchical Visual Layout

Now show me the final Google Doc exactly as it should appear when pasted.

Strict rules:
- Output ONLY the document (no explanations)
- Make formatting visually obvious and impossible to miss

Formatting system:
- Title → Centered feel using spacing + wrapped with ===
- Subtitle → Wrapped with --- directly below title
- Section headings → Start with "## " (double hash)
- Sub-points (if any) → Use "•"
- Body text → Normal paragraphs

Spacing rules:
- Keep clean spacing between every element
- Make each section clearly separated

Goal:
When I paste this into Google Docs, I should instantly see hierarchy and be able to convert:
## → Heading 1
Title → Title style
Subtitle → Subtitle style

Prompt #3: Style Marker System

Now generate the final document in a Google Docs–ready format using style markers.

Strict rules:
- Output ONLY the document
- No explanations, no notes

Formatting system (MANDATORY):
- [TITLE] Main title of the document
- [SUBTITLE] Supporting subtitle (optional, directly below title)

- [H1] Section heading
- [H2] Sub-section heading (if needed)
- [BODY] Paragraph text

- [BULLETS]
  • Point 1
  • Point 2

- [CLOSING] Final closing line

Spacing rules:
- Keep clean spacing between all elements
- Each tag must be on its own line

Goal:
This should be structured so I can quickly use “Find and Replace” in Google Docs to convert:
[TITLE] → Title style  
[H1] → Heading 1  
[H2] → Heading 2
